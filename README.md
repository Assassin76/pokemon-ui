# Pokemon UI

A Pokémon-themed user interface recreated using HTML and CSS.

## Objective

This project was created as part of Assignment 1 to practice CSS positioning concepts, specifically:

- position: relative
- position: absolute

## Features

- Pokémon card layout
- Search bar
- Filter section
- Favorite and navigation icons
- Feature cards section
- Pokémon-themed design

## Technologies Used

- HTML5
- CSS3
- Remix Icons
